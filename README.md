&lt;links /&gt;
=========

A simple as possible link shortener/tracker, meant for easy web-service integration.

(Partial) tech list:

+ Python 3
    + Flask
    + SQLAlchemy
    + WTForms
+ Postgres
+ Bootstrap 4
+ FontAwesome 4
